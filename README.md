# Logic Gate Simulator

An interactive browser-based digital logic simulator developed using HTML, CSS, and JavaScript.

## Features

- Interactive drag-and-drop logic gates
- AND, OR, NOT, NAND, NOR, XOR, XNOR
- Wire connections
- Real-time logic simulation
- Truth table generation
- Responsive interface

## Technologies

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas

## Future Improvements

- Save/Load circuits
- Half Adder and Full Adder templates
- Dark Mode
- Export truth table
- Sequential logic components
- Keyboard shortcuts

## Getting Started

Clone the repository

```bash
git clone https://github.com/<your-username>/LogicGateSimulator.git
```

Open `index.html` in your browser.
